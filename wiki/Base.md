# Base

## Overview[[edit](Base?action=edit&section=1.md)]
File:Base.png.mdBase with no player-built defences or structures

The base is a player's home point. It consists initially of a [Generator](Day_1#Generator.md) and a [Crafting Table](Crafting_Table.md). The base can be fortified and the player can gain passive heals while here. Players must return to the base at night or the poison gas will kill them.
